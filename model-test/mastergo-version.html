模型特征检测

通过输入模型参数组合与目标特征脚本，检测模型是否具备特定特征。支持多组参数和多个特征的同时检测。

参数组合脚本

def check_params():
    params = {
        'learning_rate': 0.001,
        'batch_size': 32,
        'epochs': 100
    }
    return params

目标特征脚本

def check_feature(model):
    accuracy = model.evaluate()
    return accuracy > 0.95
